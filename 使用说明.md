# 台球账目系统 使用说明

**版本：V3.0**

---

## 一、系统概述

台球账目系统是为台球器材电商公司量身打造的日常财务管理工具。核心解决两个问题：

1. **多平台卖货，钱散落在哪？** — 抖音、快手、视频号、有赞……每个平台都有钱包，提现到银行卡后又有公户、支付宝、微信等多个出口。系统帮你追踪每一笔钱的来龙去脉。
2. **账目太乱，月底对不上？** — 订单、支出、退款、转账全部记录在案，自动计算余额，月底一键看报表。

**一句话理解：这不是专业财务软件，而是一个让老板和员工都能看懂的"公司钱袋子管理器"。**

---

## 二、账户体系（核心概念）

### 两种账户

| 类型 | 说明 | 示例 |
|------|------|------|
| **平台账户** | 电商店铺的钱包，客户付款先进这里 | 抖音店铺、快手小店、视频号、有赞商城 |
| **日常账户** | 公司实际掌控的资金账户 | 公户、银行卡、支付宝、微信、现金 |

### 余额怎么算

```
账户余额 = 期初余额 + 收入 - 支出 - 转出 + 转入
```

- **收入**：订单收款进平台账户
- **支出**：进货、快递、工资等从日常账户出
- **转出/转入**：平台钱包提现到银行卡（平台转出，银行卡转入）

### 资金流向

```
客户下单付款 → 平台账户（抖音/快手等）
                    ↓ 提现
              银行卡 / 公户
                    ↓ 支出
              供应商 / 工资 / 快递 / 房租...
```

## 三、功能模块说明

### 📊 收支总览（首页）

**功能**：一目了然看公司财务全貌——本月收入、支出、退款、利润，以及各账户余额汇总。

**使用**：登录后默认进入首页，选择月份可切换查看历史数据。

**关联**：汇总了订单、支出、退款、转账所有模块的数据。

---

### 📝 订单登记

**功能**：记录每一笔客户订单。支持两种录入方式：
- **手动填写**：点"新增订单"按钮，填写客户、产品、金额、收款账户等
- **文本模式**：复制粘贴订单信息，系统自动识别解析
- **Excel 导入**：支持聚水潭、抖店导出的 Excel 文件，自动匹配字段

**使用场景**：每天营业结束后，批量录入当天所有订单。

**关联**：订单会自动更新对应账户余额，关联产品和客户信息。

---

### 💸 支出管理

**功能**：记录公司的每一笔开支。

**使用**：
1. 点"新增支出"，填写金额、类别、付款账户、备注
2. 或用**文本模式**：直接粘贴支出文本，系统自动解析（如复制银行流水）
3. 支持按类别、时间、账户筛选查看

**关联**：支出会扣减对应账户余额，在财务报表中汇总。

---

### 🔄 账户转账

**功能**：记录账户之间的资金转移，比如平台提现到银行卡。

**使用**：
1. 填写转出账户、转入账户、金额
2. 或用**文本模式**批量粘贴转账记录
3. 系统自动：转出账户余额减少，转入账户余额增加

**典型场景**：抖音店铺提现到对公户、支付宝转到银行卡等。

---

### 💳 退款管理

**功能**：记录客户退款信息，追踪退款原因和金额。

**使用**：点"登记退款"，填写退款金额、关联订单、退款原因、退款账户。

**关联**：退款会扣减对应账户余额，同时影响利润计算。

---

### 🏦 账户管理

**功能**：管理所有平台账户和日常账户，查看各账户实时余额。

**使用**：
- 新增账户：选择类型（平台/日常），填写名称和期初余额
- 查看余额：每个账户卡片显示当前余额
- 余额日志：点击账户可查看所有影响余额的收支记录

---

### 📜 余额快照

**功能**：月度对账工具。每月记录各账户余额，自动从上月结余计算期初余额。

**使用**：每月初做一次快照，核对各平台和银行卡的实际余额是否与系统一致。

---

### 📦 产品库

**功能**：管理公司销售的所有台球器材产品信息。

**使用**：添加产品名称、分类、售价等，订单登记时可直接选择产品。

---

### 👥 客户管理

**功能**：记录客户信息和历史消费记录。

**使用**：查看客户的订单历史、累计消费金额等，了解老客户复购情况。

---

### 🎧 客服号管理

**功能**：管理客服和主播账号，追踪不同客服/主播的接单业绩。

**关联**：订单可关联客服号，用于统计个人业绩和计算提成。

---

### 📈 财务报表

**功能**：生成多种报表，支持导出 Excel。

| 报表类型 | 用途 |
|----------|------|
| 收支报表 | 按月/周查看收入、支出、利润 |
| 产品销售报表 | 各产品销量和销售额排名 |
| 销售业绩报表 | 各客服/主播的业绩对比 |
| 账户余额报表 | 各账户余额汇总 |
| 提成报表 | 员工应得提成明细 |
| 现金流报表 | 资金流入流出趋势 |

---

### 📱 平台收入

**功能**：单独查看各电商平台的收入数据，按平台筛选和汇总。

---

### 📊 业绩统计

**功能**：按客服号/销售分组统计业绩数据，支持按时间段查看。

---

### 💰 提成管理

**功能**：配置提成规则，自动计算员工提成。管理员可设置不同产品的提成比例。

---

### 📊 法定财务报表

**功能**：生成利润表等标准财务报表，供财务人员使用。

**权限**：仅管理员和财务角色可查看。

---

### 🏓 教练系统

**功能**：管理台球教练信息，包括课程安排和教练档案。

**使用场景**：公司同时经营台球培训业务时使用。

---

### 🏦 股东垫资

**功能**：记录股东对公司垫付的资金，追踪借款和归还情况。

---

### 🏢 资产管理

**功能**：管理公司的固定资产（电脑、设备、车辆等），记录采购时间、原值、折旧等。

---

### ⚙️ 系统管理

| 功能 | 说明 |

| 用户管理 | 添加/编辑系统用户，分配角色和权限 |
| 销售分组 | 按团队或部门分组管理客服 |
| 系统设置 | 基础配置（仅管理员可访问） |

## 四、数据关联关系

```
客户下单 → 订单 → 关联：产品、客服号、收款账户
    ↓
订单收入 → 平台账户余额增加
    ↓
平台提现 → 转账记录 → 平台余额减少，银行卡余额增加
    ↓
日常支出 → 支出记录 → 付款账户余额减少
    ↓
客户退款 → 退款记录 → 退款账户余额减少
```

所有模块通过**账户**串联在一起，账户余额由系统自动计算，无需手动调整。

## 五、Excel / 文本导入说明

### 订单导入
支持从聚水潭、抖店导出的 Excel 文件直接导入，系统自动识别列名并匹配字段。

### 支出导入（文本模式）
在支出管理页面点击"文本模式"，将银行流水或支出记录粘贴到输入框，系统智能解析金额、日期、摘要等信息。

### 转账导入（文本模式）
在转账管理页面点击"文本模式"，批量粘贴转账记录，自动识别转出/转入账户和金额。

**提示**：首次使用导入功能时，建议先导入 1-2 条数据测试，确认解析正确后再批量导入。

## 七、常见操作 FAQ

**Q：如何录入一笔支出？**
A：进入"支出管理" → 点"新增支出" → 填写金额、选择类别和付款账户 → 保存。也可以用文本模式批量粘贴。

**Q：如何导入聚水潭订单？**
A：在聚水潭导出 Excel → 进入"订单登记" → 点"导入" → 上传文件 → 系统自动解析 → 确认导入。

**Q：如何查看某个平台的余额？**
A：进入"账户管理" → 在平台分类下找到对应店铺 → 卡片上直接显示当前余额。

**Q：如何计算员工提成？**
A：进入"提成管理" → 选择员工和时间范围 → 系统根据订单业绩和提成规则自动计算。

**Q：如何查看月度利润？**
A：进入"财务报表" → 选择"收支报表" → 选择月份 → 查看收入、支出、利润数据。也可导出 Excel。
